Projects

Mushroom Occurence In Alabama

Mushroom Occurence Chart

This project is one I'm quite fond of. I gathered species-specific data from INaturalist and imported it to Excel where I did some minor cleaning and organization of said data. I then Imported that data into Microsoft SQL Server where I cleaned it further in preparation for visualization. Finally the .csv of the cleaned table was inported to Tableau where I was able to use the geographic data to mark the locations of mushrooms. I also created sliding filters for the date of observation and the species so one can find exactly what they are looking for and see how the trends change over time.

Employee Salary Analysis Charts and Scripts

Employee Salary Analysis With Comments Explaining the Code

This project was completed entirely in Python using Jupyter Notebooks. The goal of my analysis was to find out the average salary of Data Analysts in the US based on this study and to see what the trend is based on experience level. The results were very much in line with what one would expect. From entry level to mid level, and mid level to senior, each step up provided a nicely proportional salary increase.
